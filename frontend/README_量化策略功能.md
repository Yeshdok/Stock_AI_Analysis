## 🔧 使用方法

### 启动系统
```bash
# 1. 启动后端服务
python run.py

# 2. 启动前端服务 (在frontend目录)
cd frontend
npm start
```

### 功能操作
1. **访问首页**: 打开 http://localhost:3000
2. **切换页签**: 点击顶部"股票技术分析"或"量化策略中心"
3. **浏览策略**: 在量化策略页面查看各种策略
4. **筛选策略**: 使用顶部分类标签筛选
5. **查看详情**: 点击策略卡片展开详细信息
6. **策略操作**: 使用底部按钮进行策略操作
